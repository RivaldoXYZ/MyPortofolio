IoT Engineering

I have expertise in embedded system implementation using Arduino and ESP32. In addition, I also have basic knowledge in IoT using Raspberry Pi as the center of data processing and device management. I used Raspberry Pi to process data collected from various IoT devices. I also integrated various tools for data processing, such as Mosquitto for messaging, Node-RED for automated workflows, and Grafana and InfluxDB for data visualization and storage.

Cloud Computing

I have basic skills in infrastructure management using Docker containers with the Infrastructure as Code (IaC) approach. I acquired these skills through my practicum on campus, where I applied tools such as Docker, Minikube, OpenTofu, and Ansible to support efficient infrastructure management. Although I am still a student, I have a strong commitment to continue learning and developing my skills in DevOps.

Networking

I have a basic understanding of computer networking and ethical hacking. In networking, I understand basic concepts such as OSI Layer, TCP/IP, routing, switching, and subnetting techniques. as well as VLAN, DNS, and DHCP implementation. On the ethical hacking side, I know how to identify system vulnerabilities by using tools in Kali Linux, such as Metasploit, Nmap, Burp Suite, Wireshark, and other Kali Linux tools.
